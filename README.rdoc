= web-framework-playground

Various experiments in building web frameworks in Ruby

== usage

Run a web server with a given routing strategy:

* BasicExample - <tt>bin/rake run:basic</tt>
* SinatraExample - <tt>bin/rake run:sinatra</tt> 
* HanamiExample - <tt>bin/rake run:hanami</tt>
* CampingExample - <tt>bin/rake run:camping</tt>

== Testing

  bin/rake test

== Linting

  bin/rake lint

== Documentation

  bin/rake yard
