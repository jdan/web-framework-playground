= web-framework-playground

Various experiments in building web frameworks in Ruby

== usage

Run a web server with a given routing strategy:

* BasicExample - <tt>bundle exec rake run:basic</tt>
* SinatraExample - <tt>bundle exec rake run:sinatra</tt> 
* HanamiExample - <tt>bundle exec rake run:hanami</tt>
* CampingExample - <tt>bundle exec rake run:camping</tt>

== Testing

  bundle exec rake test

== Linting

  bundle exec rake lint

== Documentation

  bundle exec rake rdoc
